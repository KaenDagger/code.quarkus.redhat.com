# Red Hat Code Quarkus App

# Backend dev

Start in watch mode:
```bash
make code.quarkus.io dev-backend
```


# Full Dev (back + front)

Start in watch mode:
```bash
make code.quarkus.io dev
```

Open: http://locahost:3000 (:8080 is the backend)


Then open `./src/main/frontend` with your favorite IDE to edit.
